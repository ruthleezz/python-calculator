# Мой первый калькулятор на Python

Простая программа, которая:
- Приветствует пользователя
- Спрашивает имя и возраст
- Предлагает угадать, что это за программа
- Выполняет основные математические операции (+, -, *, /)

## Как запустить
1. Установи Python
2. Скачай файл `calculator.py`
3. Запусти в терминале: `python calculator.py`

## Чему я научился
- Работа с `input()` и `print()`
- Условные операторы `if/elif/else`
- Преобразование типов данных
